DetSpace: Engineering detectable pathways for bio-based production

DetSpace allows the selection and design of biological circuits for sensing of bio-based chemicals by its metabolic conversion into an allosteric transcription factor-based detectable molecule. In that way, DetSpace provides a tool for precision engineering of screening and dynamic regulation of bio-based pathways of industrial interest for the bioeconomy.

The design process consists of the following steps:

  1. Select the chassis organism where the pathway will be engineered. The list contains a selection of main industrial and prototyping chassis used by the synthetic biology and metabolic engineering community. The selected chassis will determine the number of endogenous and heterologous metabolites that are present in the pathway. Typically, a DetSpace pathway should contain a high ratio of heterologoues metabolites in order to avoid cross-talk and flux diversion and, therefore, this consideration might determine in some cases the selection of the appropriate chassis.
  2. Select the target compound of interest from a list that contains a comprehensive metabolic map for production of bio-based chemicals. DetSpace provides a curated list of bio-based chemicals that are essential building blocks and compounds for the bioeconomy that is based on the metabolic map collected by S.Y Lee and colleagues in 2023. Only those compounds associated with detectable pathways are available for selection in the list.
  3. Select the effector compound of a transcription factor that can be connected to the target product. A curated list of detectable molecules that can act as allosteric effectors for transcription factors is provided based on the molecule-transcription factor pairs database collected by P. Carbonell and colleagues in 2023. For each selected target compound, only those effectors that can be associated through a DetSpace pathway to the compound are shown in the list.
  4. Retrosynthetic routes for production of the precursors in the chassis are graphically displayed, as well as required supplements to add to the growth media. The routes are calculated based on a Monte Carlo tree search algorithm that prioritizes the pathways involving less complex biochemical transformations from the reaction rules RetroRules database. Each given solution pair of target product and effectors can lead to multiple alternatives pathways. A prioritization list is provided based on the reaction rules scores. The pathways are displayed graphically in an interactive map where the user can select individual pathways and explore in more detail the intermediates in the pathway, as well as the information about each enzymatic transformation.
  5. For the target bio-based production, the main backbone upstream routes for bio-based production are detailed. Main production routes of bio-based compounds are provided based on their metabolic map, which can be further analyzed through the use of bioretrosynthesis tools such as RetroPath 2.0.
  6. DetSpace provides the list of enzyme sequences, selected through the Selenzyme algorithm and the combinatorial library of the genetic circuits for building the detectable pathway. The selected genes and pathways can be exported into SBML format so that they can be transferred to the Galaxy SynBioCAD genetic design workflow, in order to build and assembly a combinatorial library for further engineering and optimization of the DetSpace detectable pathway for bio-based production.

A walk-through tutorial

In this tutorial, we assume that the user wants to produce vanillin in the E. coli chassis (see for instance Yoon et al, Biotechnol. Lett., 2005)

API usage

DetSpace provides an API for easy access to the main functionalities of the tool. The code is freely available at the GitHub repository.


   import requests
   import urllib.parse

   url = "https://detspace.i2sysbio.uv.es"

   def version():
      """ Returns API version"""
      api_url = urllib.parse.urljoin(url,"api/version")
      response = requests.get(api_url)
      return response.json()

   def prod():
      """ Returns producibles"""
      api_url = urllib.parse.urljoin(url,"api/prod")
      response = requests.get(api_url)
      return response.json()

   def det():
      """ Returns detectables"""
      api_url = urllib.parse.urljoin(url,"api/det")
      response = requests.get(api_url)
      return response.json()

   def prod_det(det):
      """ Returns producibles connected to given detectable"""
      api_url = urllib.parse.urljoin(url,"/".join(["api/prod",str(det)]))
      response = requests.get(api_url)
      return response.json()

   def det_prod(prod):
      """ Returns detectables connected to given producible"""
      api_url = urllib.parse.urljoin(url,"/".join(["api/det",str(prod)]))
      response = requests.get(api_url)
      return response.json()
