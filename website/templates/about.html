DetSpace: Engineering detectable pathways for bio-based production

DetSpace allows the selection and design of a biological circuit that performs the sensing of a bio-based chemical by its enzymatic conversion into a allosteric transcription factor-based detectable molecule. In that way, DetSpace provides a tool for precision engineering of bio-based pathways of industrial interest for the bioeconomy.

Steps:

  1. Select the chassis organism where the pathway will be engineered.
  2. Select the target compound of interest from a list that contains a comprehensive metabolic map for production of bio-based chemicals.
  3. Select the effector compound of a transcription factor that can be connected to the target product.
  4. Retrosynthetic routes for production of the precursors in the chassis are graphically displayed, as well as required supplements to add to the growth media.
  5. For the target bio-based production, the main backbone upstream routes for bio-based production are detailed.
  6. DetSpace provides the list of enzyme sequences, selected through the Selenzyme algorithm and the combinatorial library of the genetic circuits for building the detectable pathway.

API usage

DetSpace provides an API for easy access to the main functionalities of the tool. The code is freely available at the GitHub repository.


   import requests
   import urllib.parse

   url = "https://detspace.i2sysbio.uv.es"

   def version():
      """ Returns API version"""
      api_url = urllib.parse.urljoin(url,"api/version")
      response = requests.get(api_url)
      return response.json()

   def prod():
      """ Returns producibles"""
      api_url = urllib.parse.urljoin(url,"api/prod")
      response = requests.get(api_url)
      return response.json()

   def det():
      """ Returns detectables"""
      api_url = urllib.parse.urljoin(url,"api/det")
      response = requests.get(api_url)
      return response.json()

   def prod_det(det):
      """ Returns producibles connected to given detectable"""
      api_url = urllib.parse.urljoin(url,"/".join(["api/prod",str(det)]))
      response = requests.get(api_url)
      return response.json()

   def det_prod(prod):
      """ Returns detectables connected to given producible"""
      api_url = urllib.parse.urljoin(url,"/".join(["api/det",str(prod)]))
      response = requests.get(api_url)
      return response.json()
